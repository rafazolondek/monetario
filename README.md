
# Monetário

Aplicativo desenvolvido para Android em Kotlin para converter moedas.
Projeto no qual teve a instrução do professor Tiago Aguiar do canal [Tiago Aguiar](https://www.youtube.com/@TiagoAguiar).

- Dólar
- Euro
- Peso Chileno
